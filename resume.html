Below is a list of programming languages I am proficient in coding. I list them in order from most favorite to least favorite. I don't list them here, but I am familiar with HTML and CSS programming. In my eyes, HTML is technically a language since it has language in the name (HyperText Markup Language), but it's not a challenging language to learn, so I omit it from my list. To list a few, the HTML elements should be used for their semantic meanings, inline styling should be avoided and written in CSS files, the root element should contain any shared values to avoid "duplicate code" in style sheets, and the DOM should be organized in a manner that allows for simple JavaScript code to read and manipulate it.
