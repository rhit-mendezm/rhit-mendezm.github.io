To list a few, the HTML elements should be used for their semantic meanings, inline styling should be avoided and typed in css files, the root element should contian any shared values to avoid "duplicate code" in style sheets, and the DOM should be organized in a manner that allows for simple JavaScript code to read and manipulate it.
